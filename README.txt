Проект BFSK радиомодема, реализованного с использованием Simulink, Quartus II и ModelSim
Проект включает в себя модель модема в Simulink, аппаратно-ориентированную модель модема в Simulink, результаты реализации 
модулятора и демодулятора на FPGA с использованием HDL Coder и средств синтеза Quartus, тетсбенч для проведения симуляции в ModelSim.

Основные особенности реализации модема:
- BFSK модуляция с непрерывной фазой
- Некогерентная демодуляция на основе корреляторов
- Параметризированные значения скорости передачи, частоты несущего сигнала, частоты дискретизации

Файловая структура проекта:
Описание_проекта - более подробное описание рабты модема, хода работы над проектом, результатов моделирования работы моделей и симуляции
BFSK_parameters - скрипт с инициализацией параметров моделей
BFSK_MODEM - алгоритмическая модель модема (непрерывно по времени, floating-point)
HARDWARE_FRIENDLY_BFSK - аппартано-ориентированная модель модема (дискретная, fixed-point)
run_BFSK_MODEM - скрипт с запуском моделирования BFSK_MODEM модели
run_HARDWARE_FRIENDLY_BFSK - скрипт с запуском моделирования BFSK_HARDWARE_FRIENDLY_BFSK модели

В папке BFSK_MODULATOR_HDL находятся файлы для реализации модулятора на FPGA (HDL, файлы проекта, тестбенч). Файл run в данной папке - скрипт для запуска тестбенча для модулятора в среде ModelSim.
В папке BFSK_DEMODULATOR_HDL находятся файлы для реализации демодулятора на FPGA (HDL, файлы проекта, тестбенч). Файл run в данной папке - скрипт для запуска тестбенча для демодулятора в среде ModelSim.
